## Installation

1. Clone the repository:
```bash
git clone https://github.com/skkuhg/calculator-app.git
cd calculator-app
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.
